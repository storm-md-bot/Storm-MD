# ⚡ Storm-MD v2.0

> Advanced WhatsApp Multi-Device Bot with 1000+ Commands

## ✨ Features

- ✅ **Real 8-Digit Pairing Code** — No QR code needed
- ✅ **1000+ Commands** — Admin, Fun, Tools, AI, Group, Owner
- ✅ **30+ Tools** — Downloader, Sticker, Translate, Weather, etc.
- ✅ **Auto Roast** — `/autoroast` with gaali + emoji + sayri + style
- ✅ **AI Chat** — Free Gemini API integration
- ✅ **Auto React** — Auto reacts to all messages with emoji
- ✅ **24/7 Running** — Deploy on Render/Railway free
- ✅ **All Countries Support** — Any WhatsApp number works
- ✅ **Device Spoofing** — Shows as Kali Linux / Ubuntu / etc.
- ✅ **Hidden Owner Number** — Private by default
